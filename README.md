# Go Trip - Modern Travel & Tourism Landing Page

A sleek, responsive, and visually engaging landing page for a travel agency built using **Pure HTML5 and CSS3**. This project focuses on modern UI trends like Glassmorphism, CSS Grid, and Flexbox layouts.

## 🚀 Demo
**[Click here to view the live project](https://ItzNitish01.github.io/Go-Trip/)**

## ✨ Key Features
* **Modern UI Design:** Implements Glassmorphism in the banner search form and sticky navigation.
* **Fully Responsive:** Optimized for Desktops, Tablets, and Mobile devices using CSS Media Queries.
* **Interactive Elements:** * Smooth scrolling with CSS `scroll-behavior`.
    * Creative hover effects (Card lifting, Icon rotation, Image zooming).
    * Section targeting with visual highlight animations.
* **Semantic HTML:** Built with accessibility and SEO in mind using proper header tags and descriptive sections.

## 🛠️ Technologies Used
* **HTML5:** Structural backbone and semantic layout.
* **CSS3:** Custom styling, Flexbox, Grid, and Keyframe animations.
* **Font Awesome 6:** For professional vector icons.
* **Google Fonts:** Barlow Condensed, Satisfy, and Roboto.
